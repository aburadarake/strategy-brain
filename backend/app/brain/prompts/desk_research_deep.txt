第2段階で大切なのは、自分が「面白い」と思った論点を深掘り対象として選ぶことです。
AIは示されたものは何でも深掘りしますが、どこを深掘りするかの判断は人間にしかできません。

---

## 第2段階：深掘り（焦点を絞った深掘り）

以下の【深掘り対象】について、さらに深掘りしてください。

### STEP1：歴史を3段階で俯瞰する

【深掘り対象】の歴史を3つの時代/フェーズに分けて整理してください。
- 各フェーズの特徴と代表的な出来事
- なぜそのフェーズが終わり、次のフェーズに移行したか
- それを動かした力（技術・経済・文化・規制など）

### STEP2：断絶点・境界線の整理

【深掘り対象】を変容させる可能性のある「断絶点・境界線」を整理してください。
- 変容が起きる条件とトリガー
- 変容が起きた場合のシナリオ（誰が勝ち、誰が負けるか）
- 現時点でのシグナル（すでに見え始めている変化の兆候）

### STEP3：コミュニケーション機会の特定

【深掘り対象】を踏まえると、クライアントのコミュニケーションにどのような可能性があるかを3点にまとめてください。
- それぞれの機会がなぜ「今」なのか
- どんなクリエイティブ方向が考えられるか

## 出力形式（JSON）

```json
{
  "focus": "深掘り対象のタイトル",
  "history_stages": [
    {
      "period": "時代/フェーズ名",
      "description": "このフェーズの特徴と出来事",
      "why_it_changed": "なぜ次のフェーズに移行したか"
    }
  ],
  "disruption_points": [
    {
      "title": "断絶点のタイトル",
      "description": "この断絶点の内容",
      "scenario": "変容が起きる場合のシナリオ"
    }
  ],
  "opportunities": [
    "コミュニケーション機会1（なぜ今なのか + クリエイティブ方向を含む）",
    "コミュニケーション機会2",
    "コミュニケーション機会3"
  ]
}
```

必ず上記のJSON形式で回答すること。
