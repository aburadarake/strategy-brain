# Strategy Brain — 統合プロンプト集

商品名を入れるだけで動くプランニングツール「Strategy Brain」で使用している全プロンプトをまとめたドキュメントです。
各プロンプトは単体でChatGPT/Claude等に貼り付けて使うこともできます。

---

## 実行順序と全体像

```
[入力] 商品名（＋オプションでオリエン資料）
    ↓
① 細田式3Dモデル   ← 問いのリフレーミング
    ↓
② 障壁分析STEP1   ← 使わない理由30項目
    ↓
③ 障壁分析STEP2   ← 因果関係50個
    ↓
④ 障壁分析STEP3   ← ABC分類（サービス/広告/PR）
    ↓
⑤ WHO分析 ＋ WHAT分析（並列）
    ↓
⑥ BIG IDEA生成
    ↓
⑦ コピーライティング（10案）＋ 広告企画（6案）（並列）
```

## PROMPT 1: 細田式3Dモデル（問いのリフレーミング）

> **目的**: ビジネス課題を「人間の課題」に変換し、凡庸な正解を超えた可能性を解放する
> **元ネタ**: TBWA\HAKUHODO 細田高広氏の思考メソッド

```
【製品・サービス名】の戦略を考えたいです。
以下の3Dモデルで分析してください。

## STEP 1: DOUBT（疑う）
提示されたビジネス課題に対して、5つの「疑い」を投げかけてください：
- その目標設定自体が本当に正しいのか？
- 前提となっている思い込みは何か？
- 「業界の常識」として疑われていないものは何か？
- AIや競合が同じ答えを出す「凡庸な正解」になっていないか？
- 本当の問題は、表面に見えている問題とは別のところにないか？

## STEP 2: DISCOVER（発見する）
DOUBTを踏まえ、ビジネス課題を「人間の課題」にリフレーミングしてください：
- 「売上を上げる」→「なぜ人々はこれを選ばないのか？」
- 「認知を上げる」→「なぜ人々の心に残らないのか？」
- 「ブランド価値を上げる」→「なぜ人々はこれを愛せないのか？」

## STEP 3: DESIGN（デザインする）
DISCOVERで見出した課題に対して、3つの「可能性解放アイデア」を提案してください。
各アイデアは「〜を可能にする」という形式で。なぜ凡庸でないかも説明してください。
```

## PROMPT 2: 障壁分析（STEP 1〜3）

> **目的**: 「使わない理由」の本質的な課題を見つけ、解決策の分類まで行う

### STEP 1: 使わない理由30項目の抽出

```
【製品・サービス名】を使わない理由を、以下の観点から30項目出してください。
番号を振り、各項目にカテゴリ（製品要因/心理要因/社会的要因/文化・慣習）も付けてください。

観点：
1. 製品要因（機能、価格、品質、使いやすさ）
2. 心理要因（不安、面倒くささ、習慣、認知バイアス）
3. 社会的要因（周囲の目、社会的証明の不足）
4. 文化・慣習（既存の習慣、世代間ギャップ）
※ 社会的要因・文化/慣習で「はじめられない理由」も必ず含めること
```

### STEP 2: 因果関係の整理（50個）

```
上記の30項目について、因果関係がある/ありそうなものを50個程度整理してください。
形式: 「【障壁A】が原因で【障壁B】が起きる」

加えて、「つながりが多い重要な障壁」（5〜10個）を特定して教えてください。
つながりが多い障壁は、そこを解消すれば連鎖的に他の障壁も崩れる「根本原因」です。
```

### STEP 3: ABC分類と解決アプローチ

```
上記の障壁を以下のABC基準で分類し、解決アプローチを提案してください：

A（サービスで解決できるもの）
→ 機能追加・UX改善・価格変更など、プロダクト自体の改善で解決できるもの

B（広告やブランディングで解決できるもの）
→ 認知向上・イメージ変容・メッセージング改善で解決できるもの

C（PRなどで社会におけるあり方を変えるもの）
→ 社会的な認識変容・文化変革が必要なもの

各分類に対して、具体的な解決アプローチも提案してください。
```

## PROMPT 3: WHO分析（消費者インサイトの深掘り）

> **目的**: 表層的なデモグラを超えて、消費者の「本音・葛藤・隠れた動機」を把握する

```
【製品・サービス名】のWHO分析を行ってください。

## 1. コアターゲット解剖
- 表層的特性（デモグラフィック）
- 行動特性（購買行動/メディア接触）
- 価値観と信念体系
- 未充足ニーズ
- 隠れた感情・動機
- 購買決定に影響する心理的トリガー

## 2. ターゲットセグメンテーション
- プライマリターゲット（最優先）
- セカンダリターゲット（拡大）

## 3. 消費者インサイト（最低3つ）
各インサイトについて以下を明記：
- インサイト文（「〜と思っているが、実は〜」という形式）
- 抱える矛盾・葛藤（テンション）
- ブランドが入り込む機会

## 4. 未充足ニーズ
現在の市場で満たされていないニーズを列挙
```

## PROMPT 4: WHAT分析（ブランド戦略）

> **目的**: 市場環境を整理し、差別化できるポジションと提供価値を明確にする

```
【製品・サービス名】のWHAT分析を行ってください。

## 1. 市場環境分析
- 市場概況（規模、トレンド、成長性）
- 競合状況（主要プレイヤー、ポジショニング）
- ポジショニング機会（空白地帯、差別化できる領域）

## 2. ブランド診断
- 強み（競争優位性、独自アセット）
- 弱み（課題、改善すべき点）
- 認識ギャップ（提供価値と消費者認識の乖離）

## 3. 障壁打破戦略
各障壁タイプ別の具体的解消方法：
- 機能的障壁: 原因と解消方法
- 経済的障壁: 原因と解消方法
- 心理的障壁: 原因と解消方法
- 社会的障壁: 原因と解消方法

## 4. 提供価値の再定義
- 現在の価値認識: 「○○○○」
- 移行すべき新価値認識: 「○○○○」
- 理想的な最終価値認識: 「○○○○」
- 機能的価値 / 情緒的価値 / 社会的価値 / コアプロポジション

## 5. 戦略的差別化要素（機能的・感情的それぞれ3つ）
競合との本質的な差異を浮き彫りにすること
```

## PROMPT 5: BIG IDEA生成

> **目的**: WHO×WHATから、時代を超えて響くブランドの核心的アイデアを生み出す

```
以下の情報を踏まえて、【製品・サービス名】のBIG IDEAを生成してください。

[WHO分析の要約を貼り付け]
[WHAT分析の要約を貼り付け]

## BIG IDEAの5つの評価基準
| # | 基準 | 問い |
|---|------|------|
| 1 | シンプル（Simplicity） | 一言で伝わるか？ |
| 2 | インサイト（Insight） | 消費者の深い欲求を突いているか？ |
| 3 | 感情（Emotion） | 人の心を動かし、共感を生むか？ |
| 4 | 普遍性（Timeless） | 時代や市場が変わっても通用するか？ |
| 5 | 行動（Actionable） | 消費者が具体的に何をすべきかを示せるか？ |

## ワークシート（以下を埋めてBIG IDEAを導く）
1. ブランドの圧倒的真実（Core Truth）は何か？
2. ターゲットの深層的な願望・問題は何か？
3. 想起させたい感情は？
4. 10年後も通用するか？
5. 「〜な人のための〜」で5〜7語で表現すると？

## 出力
- BIG IDEA（5〜7語のシンプルな表現）
- 戦略的根拠（なぜこれなのか）
- 5基準での評価（各1〜5点）
- 代替案3つ
```

## PROMPT 6: 戦略的コピーライティング（10案）

> **目的**: BIG IDEAをベースに、表現技法を駆使した記憶に残るコピーを生成する

```
以下の情報を踏まえて、戦略的なコピーを10案生成してください。

[BIG IDEA、WHO、WHAT の要約を貼り付け]

## コピーを書く前に必ず考えること（2段階の深掘り）

### GOAL｜目的
- 最終的に人をどう動かすか？（行動/認知/感情）
- 「感情変容」もセットで言語化する

### WHO｜ターゲットの葛藤
- 「いま何に葛藤しているか」を描写する
- 「なぜその葛藤を放置しているのか」まで掘り下げる

### VALUE｜価値ある事実
- 意外だが真実な事実を3つ以上抽出する
- 「見え方が180度変わる」1点を選ぶ

## 使える表現技法
【スタイル】感情型 / シンプル型 / ストーリー型 / 造語型 / ビジネス信頼型

【テクニック】
- 大きく言う（人生・世界の視点）
- 常識を覆す（逆説・挑戦）
- スケールギャップ（大小対比）
- 肯否を入れる（できない/できるの組み合わせ）
- 逆の言葉（対極キーワードを並べる）
- 感嘆詞を入れる（「そうだ」「あ」など）
- ストーリー性を持たせる（情景描写）
- 言葉を掘り下げる（概念の再定義）

【構文パターン】
- 「AはBでできている」型（例：世界は誰かの仕事でできている）
- 「AはBにある」型（例：愛は食卓にある）
- 「AがBになる」型（例：一言が、愛になる）
- 「〇〇だけで」型
- 「〇〇を向く」型
- 「同じ言葉を重ねる」型
- 「動詞＋倒置法」型
- 「本当に〇〇なのは」型

## 鉄則
- "言葉だけ面白いコピー"は不可。「行動につながる認知変容」を生む
- 「自分がその人だったら、グサッと刺さるか？」を常に問い直す

## 出力形式
各案: ヘッドライン / 使用テクニック / 解説 / 推奨案と理由
```

## PROMPT 7: 広告企画 6案（発想法別）

> **目的**: 6つの異なる発想法から、方針の違う広告企画を立案する

```
以下の情報を踏まえて、6つの発想法それぞれを用いた広告企画を1案ずつ生成してください。

[製品情報、WHO、WHAT、BIG IDEA を貼り付け]

## 6つの発想法

① ナンバーワン法
「この点だけは世界一/日本一/唯一」という優位性を発見し、そこを軸に企画を立てる

② バズ/話題化法
「これ、友達に教えたい！」という拡散の仕掛けを設計し、口コミを起点に認知を広げる

③ 真反対法
商品特有の「当たり前」の真逆を設計して、新しい文脈価値をつくる
例: ガリガリ君（安く雑に食べる）→「ガリガリ君、コースで。」（高級フレンチ）

④ 極端法
商品の特徴を"ありえないほど"誇張してバズや記憶に残す
例: 「一瞬で終わるバイト」（マッハバイト）

⑤ 体験価値法
機能より「体験後の感情・変化」に焦点を当て、購入前から体験を届ける

⑥ 共感/インサイト法
「言葉にできなかった感情や本音」を言語化し、「そう、これだよ！」という共鳴を作る

## 各案に含めること（8項目）
1. 企画名（短く覚えやすく）
2. 使用した発想法
3. コアメッセージ（1行で言い切る）
4. 企画の仕組み（どう広がるか）
5. OOH広告コピー案 5本（一瞬で理解できる短いコピー）
6. SNS投稿フォーマット 2つ
7. 体験/プロダクト/イベント/PR施策 1つ
8. 成功条件 + KPI例

## また、企画前提として以下も設計すること
- ブランドコンセプト（一言）とその背景ストーリー
- 商材に対する新しい視点・気づき（5項目以上）
  → 今まで語られていなかった魅力、生活シーンの盲点、五感的価値など
```

## よく使う組み合わせ（ショートカット）

### パターンA: 商品名だけで全部
```
【製品名】について、以下をすべて実行してください：
1. 細田式3Dモデル（Doubt/Discover/Design）
2. 使わない理由30項目＋因果関係＋ABC分類
3. WHO分析（インサイト3つ以上）
4. WHAT分析（価値の再定義まで）
5. BIG IDEA（代替案3つ含む）
6. コピー10案（技法・スタイル多様に）
7. 広告企画6案（発想法別）
```

### パターンB: インサイト+コピーだけ素早く
```
【製品名】について：
- ターゲットの本質的な葛藤を3つ抽出
- 各葛藤に対応するインサイト発のコピーを各3本（計9本）生成
- 最も刺さる1本を推奨して理由を説明
```

### パターンC: 広告企画だけ
```
【製品名】の広告企画を、①ナンバーワン法 ②バズ法 ③真反対法 の3手法で1案ずつ生成してください。
各案にOOHコピー5本、SNS案2つ、KPI例を含めてください。
```
